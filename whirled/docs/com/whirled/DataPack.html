getDisplayObjects()method 
public function getDisplayObjects(names:Array, callback:Function, useSubDomain:Boolean = false):void

Get some display objects in the datapack.

Parameters
names:Array — an Array of the names of the display objects to load.
 
callback:Function — a Function that will be called when all the display objects are loaded (or were unable to load). Signature: function (results :Object) :void results will contain a mapping from name -> DisplayObject, or null if none.
 
useSubDomain:Boolean (default = false) — if true, classes in a loaded SWF will be added to an ApplicationDomain that is a child of the current ApplicationDomain.
