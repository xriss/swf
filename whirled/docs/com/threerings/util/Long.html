public static function fromNumber(value:Number = 0):Long

Creates a new Long from the provided variable. Only integers in the [-2^63, 2^63) range can be converted; non-integer values in this range will be rounded, and values outside of the range will trigger an ArgumentError. Additionally, since Number is a double-precision floating point value, values outside of the [-2^52, 2^52) range will suffer loss of precision.
