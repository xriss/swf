encode()method 
public static function encode(obj:Object, encodeArrayElements:Boolean = false):Object

Encode the specified object as either a byte[] or a byte[][] (see below). The specific mechanism of encoding is not important, as long as decode returns a clone of the original object. No validation is done to verify that the object can be serialized. Currently, cycles in the object graph are preserved on the other end.

Parameters
obj:Object — if true and the obj is an Array, each element is encoded separately, returning a byte[][] instead of a byte[].
 
encodeArrayElements:Boolean (default = false)
