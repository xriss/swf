Classpublic class EmbeddedSwfLoader
InheritanceEmbeddedSwfLoader flash.events.EventDispatcher

Allows you to load an embeded SWF stored as a Byte Array then access any stored classes within the SWF. Embed your swf like: [Embed(source="foo.swf", mimeType="application/octet-stream")] Then, instantiate that class and pass it to load() as a ByteArray.
